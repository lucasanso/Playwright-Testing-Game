# Teste de palavras no jogo Contexto (web)

Nesse script eu implementei uma digitação automática de palavras no jogo [Contexto](https://contexto.me/pt/)  com o simulador de navegador Playwright.

### Criando ambiente de desenvolvimento
```bash 
$ python3 -m venv venv && source venv/bin/activate
```

### Instalando dependências
```bash
$ pip install -r requirements.txt
```

### Executando o simulador
```bash
$ python3 tester.py
```
